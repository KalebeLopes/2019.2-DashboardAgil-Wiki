# DashOwl

DashOwl é um dashboard ágil criado com o intuito de acompanhar em tempo real os repositórios do GitHub tirando métricas e indicadores das atividades feitas individualmente e pelo time. O projeto é uma extensão do GitHub no navegador que, permite aos usuários que utilizam a plataforma GitHub ao entrar em certo repositório tenham a opção de ver graficamente de acordo com a metodologia ágil um breve histórico do que foi feito e/ou está sendo feito atualmente no projeto.
